##  Descripción

**T-Under** es una aplicación de **e-commerce moderna** desarrollada con **React** y **Vite**, que permite explorar productos de forma dinámica, realizar búsquedas avanzadas y gestionar un carrito de compras de manera intuitiva.  

El proyecto consume los datos de productos desde la **API pública Edukuk**, mostrando fotos, precios y descripciones de manera atractiva y responsiva.  

---

##  Funcionalidades principales

- **Listado de productos** con imágenes y descripciones.
- **Buscador** en tiempo real que permite filtrar productos por nombre o descripción.
- **Filtros y categorías** para ordenar y mostrar productos según tipo y público.
- **Ordenar productos** por relevancia o precio (menor a mayor / mayor a menor).
- **Agregar productos al carrito** y **eliminación de productos**.
- **Carrito dinámico** mostrando cantidad, total y control de productos.
- **Diseño responsive**: adaptado a escritorio, tablet y móviles.
- **Efecto Masonry** para un grid de productos moderno y visualmente atractivo.

---

##  Tecnologías utilizadas

- **React 18**
- **Vite** como bundler y entorno de desarrollo rápido.
- **React Router DOM** para navegación SPA.
- **React Context** para manejo de estado global (carrito de compras).
- **react-masonry-css** para el grid tipo Pinterest.
- **CSS moderno y responsive** con Flexbox y Grid.
